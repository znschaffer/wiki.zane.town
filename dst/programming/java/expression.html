Expression

A simple value or a set of operations that produces a value.

Example: 42 or 28.9

Evaluation
The process of obtaining the value (result) of an expression.
Operator
A special symbol that is used to indicate an operation to be performed on one or more values.

Literals

Simple values used in expressions.

  • Integer literal: 432
  • Double literal: 21.34
  • String literal: "Hi There"
  • Character literal: 'R'
  • Boolean literal: true
